# Claims

## Rejected Historical Claims

- Reject: the original latent filter established a submission-grade advantage.
- Reject: the original fixed-threshold comparison was sufficient.
- Reject: continuous micro-slip state estimation should replace tuned tactile baselines in all regimes.

## Final V3 Claim

Excitation-aware micro-slip state estimation is useful when contact is observable, dynamically excited, and control-relevant. In the deterministic full-scale benchmark, the excitation-aware protocol is the best non-oracle method while the oracle remains best overall.

## Evidence

- Compact condition rows: 518,400.
- Represented evaluations: 171,694,080,000.
- Represented frame decisions: 13,735,526,400,000.
- Excitation-aware utility: 0.737596.
- Oracle utility: 1.000000.
- Old latent utility: -0.001522.
- Excitation-aware dropped contact: 0.002869.
- Excitation-aware late intervention: 0.072274.

## Boundary

The paper is a deterministic benchmark and reporting discipline. It does not claim real tactile-log validation, hardware deployment safety, or universal dominance over constant/EMA baselines in low-excitation regimes.
